# rschrodinger: Fast Data Bridge Between Schrodinger and R

Read Maestro (.mae/.maegz) files natively in R without requiring a
Schrodinger installation. Optionally bridge to Schrodinger's Python API
for protein-ligand interaction analysis and job monitoring via sandboxed
subprocess calls.
